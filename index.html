Conheça mais sobre a aplicação Busca de Perfis do Github

O que é?

Uma aplicação simples e rápida para buscar perfis públicos do GitHub
e explorar informações relevantes como repositórios, seguidores,
seguindo e muito mais.

Funcionalidades

Buscar perfis públicos do Github

Visualizar informações principais do usuário

Listar repositórios e linguagens utilizadas

Ver números de seguidores e seguindo

Acesso rápido ao perfil do Github

Sobre os dados

Todos os dados exibidos são obtidos da API pública do Github e
estão sujeitos aos termos de uso do Github.
